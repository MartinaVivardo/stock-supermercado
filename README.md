Sistema de Control de Stock para Supermercado

Aplicación web desarrollada con **HTML, CSS y JavaScript** que permite gestionar el inventario de productos de un supermercado. Pensado para carga rápida, control visual del stock y administración básica sin necesidad de base de datos externa (usa LocalStorage).
Funcionalidades principales
✅ Agregar, editar y eliminar productos  
✅ Ajustar stock (entrada y salida)  
✅ Filtrado por categoría, proveedor y estado  
✅ Búsqueda inteligente (nombre, proveedor, código)  
✅ Alerta visual de stock bajo o sin stock  
✅ Exportación e importación CSV  
✅ Estadísticas automáticas:
- cantidad total de productos
- valorización en costo
- valorización en precio de venta
- productos en stock bajo

✅ Click sobre Categoría o Proveedor para filtrar automáticamente  
✅ Multi-selección de filas para acciones masivas 

 Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- LocalStorage (persistencia de datos)
- Blob API (exportación CSV)
